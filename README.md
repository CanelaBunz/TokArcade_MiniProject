# TokArcade 🐱🎮

¡Bienvenido a **TokArcade**! Esta es una mini-aplicación de juegos diseñada para integrarse con sistemas de Mini Programas (como TokaPay/Alipay). Aquí encontrarás diversión rápida y adictiva con nuestro gato protagonista, Tokat.

---

## 🕹️ Nuestros Juegos

Hemos diseñado tres experiencias únicas para que nunca te aburras:

1.  **Tokat Says**: Un duelo de memoria. Mira los movimientos de Tokat y repítelos sin equivocarte. ¿Hasta qué ronda puedes llegar?
2.  **Toka RPS Rhythm**: El clásico "Piedra, Papel o Tijeras" pero llevado al siguiente nivel. Debes seguir el ritmo de la música y tocar en el momento exacto para vencer a Tokat.
3.  **Detective Tokat**: Un juego de ingenio donde deberás resolver acertijos y encontrar pistas usando tu linterna. ¡Demuestra que eres el mejor detective!

---

## 🚀 ¿Qué hace especial a TokArcade?

*   **Paga para Continuar**: Si te quedas sin tiempo o pierdes, ¡no te rindas! Puedes pagar un centavo ($0.01 MXN) para obtener una vida extra o tiempo adicional y seguir rompiendo récords.
*   **Modo Resiliente (Demo Mode)**: Sabemos que a veces el internet falla. TokArcade está diseñada para ser "a prueba de fallos". Si el servidor no responde, la aplicación activa automáticamente un **Modo Demo** que te permite seguir jugando sin interrupciones.
*   **Diseño Premium**: Interfaz moderna, animaciones fluidas y colores vibrantes diseñados para que la experiencia sea increíble en tu celular.

---

## 🛠️ Detalles Técnicos (Simple)

Para los curiosos o desarrolladores:
*   **Plataforma**: Alipay Mini Program Framework.
*   **Lenguajes**: JavaScript (Lógica), AXML (Estructura) y ACSS (Estilo).
*   **Arquitectura**: Usamos un diseño limpio (Clean Architecture) para que el código sea fácil de entender y mantener.
*   **Conectividad**: Se comunica con la API de Toka en Azure para manejar pagos y usuarios de forma profesional.

---

## 💻 ¿Cómo ejecutarlo?

1.  Descarga el **Mini Program Studio** (IDE de Alipay).
2.  Importa esta carpeta como un nuevo proyecto.
3.  Haz clic en el botón de **Simular** o **Remote Debug** para empezar a jugar.
